## Tools & Frameworks
```
- Flask v1.1.2
- Bootstrap v4.5
- Jinja Template Engine
- SQLite3
- SQLAlchemy
```

## Features
```
- User login/logout
- Password hashing
- New user registration
- Items buying/selling
- Easily customizable
- Modern interface
```

## Setup
```
git clone https://github.com/digamjain/Flask-Website
cd Flask-Website
pip install -r requirements.txt
python3 run.py
```
